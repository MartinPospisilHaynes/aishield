# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅         |

## Reporting a Vulnerability

Pokud najdete bezpečnostní problém, **NEposílejte** ho jako veřejný Issue!

Kontaktujte nás přímo:
- **Email**: info@aishield.cz
- **Předmět**: [SECURITY] Popis problému

Odpovíme do 48 hodin.

## Scope

- aishield.cz (frontend)
- api.aishield.cz (backend API)
- Supabase database
- Autentizace & JWT
